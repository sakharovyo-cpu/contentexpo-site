# Video Production Landing (с видео в заголовке)

## Что внутри
- `index.html` — одностраничник
- `styles.css` — стили (dark / cinematic)
- `script.js` — меню, анимации, копирование текста
- `assets/Showreel.mp4` — ваше видео для заголовка
- `assets/poster.jpg` — постер (первый кадр), используется как fallback

## Как запустить локально
Просто откройте `index.html` в браузере **или** поднимите локальный сервер:

### Вариант 1 (Python)
```bash
python -m http.server 8080
```
Откройте `http://localhost:8080`

### Вариант 2 (VS Code)
Расширение **Live Server** → *Go Live*.

## Как поменять контент
1) Заголовок, описание, кнопки — в `index.html` (секция HERO).
2) Блок "Портфолио" — замените карточки на свои превью и ссылки.
3) Контакты — замените ссылки в секции `#contact`.

## Если вы поменяете видео
Положите файл в `assets/` и обновите путь в `index.html`:
```html
<source src="assets/ВАШ_ФАЙЛ.mp4" type="video/mp4" />
```
